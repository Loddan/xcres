Usage:
    xcresources [OPTIONS] SUBCOMMAND [ARG] ...

Parameters:
    SUBCOMMAND                    subcommand
    [ARG] ...                     subcommand arguments

Subcommands:
    build                         Build the resources index files
    version                       Show the gem version

Options:
    -h, --help                    print help
    --silent                      Show nothing
    --[no-]ansi                   Show output without ANSI codes (default: true)
    -v, --verbose                 Show more debugging information
